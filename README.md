# Html-And-Css
## Features:

1. **Centered Image with Filters**: An image is centered, with a grayscale and sepia filter, and rounded edges.
2. **Responsive Navbar**: A fixed navbar that includes links to different sections of the page.
3. **Table with Skills**: A skills table with borders, padding, and a colored header.
4. **Two-Column Layout**: A portfolio section with two columns showcasing projects.
5. **Text Over Image/Video**: Overlays are used for images and videos with hover and fade effects.
6. **Autoplaying Video**: A short video embedded with autoplay, mute, and loop functionalities.
